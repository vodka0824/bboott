# 哭PC工具人 系統完整說明手冊

*最後更新：2026-06-19*

---

## 1. 系統架構與權限概要

本機器人採用模組化設計，功能透過 `Handlers` 處理，由 `Routes` 進行指令路由。

### 權限分級

| 等級 | 說明 |
|---|---|
| 一般用戶 | 可使用公開查詢功能（油價、匯率等） |
| 已授權群組成員 | 群組完成註冊後，可使用娛樂、經濟遊戲等功能 |
| 群組管理員 | 可開關群組內功能模組 |
| 系統管理員 | 擁有跨群組管理權限 |
| 超級管理員 | 最高權限，可產生註冊碼、管理全系統 |

### 3.2 搶劫系統

| 指令 | 說明 |
|---|---|
| `搶劫 @對象` | 對指定玩家發動搶劫 |
| `確認搶劫` | 確認高風險搶劫（目標為警察/議員/黑幫高層時） |

**搶劫規則：**
- 冷卻時間：2 小時（黑幫小弟 -10%、幹部 -30%、老大 -50%）
- 警察、軍人無法搶劫
- 有醫療負債時無法搶劫
- 搶劫成功後須支付 20% 洗錢手續費
- 成功增加通緝值 +5%（黑幫幹部 +3.5%、老大 +2.5%）

**搶劫結果種類：**
- 💨 殘影閃避：對方靠翅膀閃躲
- 🚨 搶劫反殺：對方還手，你損失所有金幣
- 🕶️ 黑幫威壓：惹到黑幫高層，強制賠保護費
- 🚓 遭到逮捕：被警察逮，加刑期
- 🍀 極限逃脫：幸運女神庇護，逃脫被捕
- 🦹 搶劫成功：成功掠奪目標資產
- 📸 抓到把柄：勒索議員成功

### 3.3 職業系統

> [!IMPORTANT]
> **單一職業限制 (雙職業禁止)**
> 遊戲系統嚴格禁止「雙職業」或「公職兼任」。
> 玩家在任何時刻只能擁有【警察】、【黑幫】、【議員】、【軍人】、【出家人】、【台積電】其中**一種**身分。若想轉換跑道，必須先進行對應的退出動作（例如：辭職、退伍、斷指、卸任、還俗），否則系統將會自動擋下報考與轉職。

#### 一般市民
- 無特殊技能，但安全穩定
- 可從事各種日常活動

#### 🚔 警察
| 技能 | 指令 | 說明 |
|---|---|---|
| 逮捕犯人 | `逮捕 @對象` | 對通緝犯發動逮捕 |
| 臨檢 | `臨檢 @對象` | 突擊搜查對象 |
| 攻堅賭場 | `攻堅 @對象` | 強制突入賭局 |
| 貪污起訴 | `起訴 @對象` | 以貪污罪名起訴對象（與逮捕共用 CD） |
| 吃案收賄 | `吃案` | 收受賄賂（無冷卻限制） |

**警察晉升：**
- 積累足夠警察功勳 → 晉升警察局長
- 局長擁有更短的技能冷卻時間

#### 🔫 黑幫
| 指令 | 說明 |
|---|---|
| `加入黑幫` | 成為黑幫小弟（需通緝值 ≥ 20%） |
| `退出黑幫` / `金盆洗手` | 退出黑幫（需斷指支付 50% 資產，最低 5000 萬，並陷入 24H 虛弱狀態） |
| `圍事` | 打手技能：向平民收保護費 (成功通緝值 +2%) |
| `收保護費` | 堂主技能：勒索單一議員 (成功通緝值 +5%) |
| `勒索 @對象` / `勒索政客` | 老大技能：勒索所有議員 (成功通緝值 +10%) |
| `暗殺警察 @對象` | 老大技能：耗資 1 億，30% 機率將目標警察強制革職住院 |
| `暗殺老大 @對象` | 挑戰現任老大篡位（需黑幫成員） |

**黑幫階級與江湖聲望制：**
階級判定改為綜合考量「近期活躍度 (通緝值)」與「歷史資歷 (前科數量)」。
`江湖聲望 = (通緝值 × 100) + (前科數量 × 5)`

| 階級 | 條件 | 特權 |
|---|---|---|
| 小弟 (thug) | 加入黑幫 | 搶劫 CD -10%、越獄率 +30% |
| 打手 (enforcer) | 江湖聲望 ≥ 20 | 基礎黑幫戰力 |
| 堂主 (capo) | 江湖聲望 ≥ 60 | 搶劫 CD -30%、越獄率 +50%、賭場圍事（查水表率 -50%） |
| 老大 (boss) | 聲望最高且 ≥ 100 | 搶劫 CD -50%、越獄率 100%、監獄是我家 |

#### 🏛️ 議員
- 透過選舉成為市議員
- 擁有「施壓」技能：每任期可用 1 次無視刑期出獄（通緝值歸零）
- 搶劫成功率大幅提升，且可規避刑事處分（司法保護傘）

#### 🪖 軍人
- 透過 `簽下去` 入伍，服役期間禁止搶劫、無法被搶、免受黑幫侵擾。
- **階級系統**：入伍次數即為軍階（1次=二兵...15次=少將...18次=四星上將）。
- **長官福利**：升至將官（少將以上）服役時間延長為 24 小時，退伍時可領取「終身俸」（`領終身俸`）。
- **軍中惡搞**：服役期間可進行拔草、掃地、打靶等小遊戲獲得微薄薪餉。
- **高階將軍特權**：
  - **發動戰爭** (`發動戰爭`)：【四星上將專屬】可發動對外戰爭。勝則晉升「五星上將」，敗則降級，並有 24 小時冷卻。
  - **研發軍火** (`研發軍火`)：【五星上將專屬】開啟地下軍火庫選單。可走私零件、輕兵器、重武器或特殊武器以獲取暴利。若遭查獲將直接面臨重度刑期與降級。

#### 📿 出家人 (和尚/尼姑)
- 透過 `/出家` 指令遁入空門（需捐出所有資產並沒收裝備），無前科與通緝值限制。
- 慈悲為懷：禁止搶劫、禁止參與賭博，同時也**免疫任何人對你發起的搶劫**。
- **每日簽到加成**：每日簽到時，每 1 名信徒可額外獲得 100 萬的「法師供養金」。
- **階級與業障**：分為「掃地沙彌、知名知客僧、斂財住持、邪教教主」。若「業障」累積過高，將隨機觸發天譴，沒收全數資金並流失一半信徒。

|---|---|---|

#### 🧑‍💻 台積電輪班星人
- 透過 `入職台積` 指令賣身進入護國神山，將被關在無塵室內，**全面禁止賭博（無法進入賭場）**。
- **每日分紅 (簽到)**：每日簽到時必定獲得 **2,000,000 (200 萬) 底薪**，並根據累積的 **KPI** 每點獲得 **500,000 (50 萬) 分紅**。領取後 KPI 將歸零。
- **機台警報與被動結算**：群組任何人講話都有機率觸發「機台故障警報」（台積電員工 1%、其他人 3%）。警報出現後，所有員工必須在 10 分鐘內於群組內輸入「系統隨機發出的維修指令」。10分鐘後系統將會被動結算：
  - 30 秒內秒解：KPI +50
  - 1 分鐘內解：KPI +20
  - 5 分鐘內解：KPI +5
  - 10 分鐘內慢半拍：KPI -10
  - 漏修 (Down 機)：KPI -30
- **考績吃丙**：若結算時 KPI 小於 0，將無法獲得分紅，並掛上嘲諷稱號。
- **主動技能**：
  - `放乖乖`：(冷卻 2 小時) 啟動後 30 分鐘內，若發生機台警報，系統將直接判定為 30 秒內秒解 (獲得 50 KPI)。但有 5% 機率手殘放成黃色乖乖，直接倒扣 30 KPI！
  - `加班`：(冷卻 30 分鐘) 增加自身疲勞值，使得下一次成功維修的 KPI 獎勵暴增 3 倍！但疲勞值越高，越容易觸發「爆肝送醫」：強制扣除總資產 50% 作為醫藥費並昏迷 6 小時。
  - `甩鍋 @對象`：(冷卻 24 小時) 只能在漏修扣績效時使用。將 100 萬罰單推給目標對象，並回復自身 20 點 KPI。
- 透過 `離職` 指令脫離苦海，所有未結算的 KPI 將全數化為烏有。

### 3.4 通緝系統

通緝值範圍：**0% ~ 100%+（可無上限累加）**

| 通緝值門檻 | 效果 |
|---|---|
| 達到 **50%** | 🕶️ 黑幫「威壓護體」啟動，免疫一般市民搶劫 |
| 達到 **60%** | 🎰 黑幫「賭場圍事」生效，查水表機率減半 |
| 達到 **100%+** | 🚨 頭號通緝犯，賭場被抓機率大增 |

**通緝值增加來源：**

| 行為 | 通緝增加量 |
|---|---|
| 搶劫成功 | +5%（黑幫幹部 +3.5%、老大 +2.5%） |
| 越獄成功 | 依剩餘刑期計算（見下方說明） |
| 勒索成功 | +15% |
| 其他犯罪 | 依行為而定 |

**通緝值歸零方式：**

| 方式 | 條件 |
|---|---|
| 捐款贖罪 | 花費哭幣按比例消除通緝值 |
| 抄寫佛經 | 每次降低少量通緝值（冷卻時間長） |
| 遭警察逮捕 | 服滿刑期後，通緝值依違規項目部分扣除 |
| 被暗殺老大成功篡位 | 舊老大通緝值歸零 |
| 議員施壓（自己） | 通緝值歸零 + 免刑 |

### 3.6 越獄系統詳細說明

> **更新日期：2026-06-19**

#### 越獄成功率
```
基礎成功率 = 5% + (EVA 數值 × 1.0625%)

黑幫階級加成：
  小弟：+30%
  幹部：+50%
  老大：100%（必定成功）

特殊道具：
  銼刀（hasShiv）：必定成功，使用後消耗
```

#### 越獄失敗懲罰
- 刑期增加 **+60 分鐘**
- 觸發 **10 分鐘冷卻**（期間不可再次越獄）
- 多次在冷卻期間亂按：額外加刑 +5 分鐘 + 冷卻延長 +5 分鐘

#### 越獄成功後的通緝值計算（方案D）

> **設計重點：防止黑幫利用越獄刷通緝值**

```
通緝增幅 = 越獄時剩餘刑期（分鐘） ÷ 300

職業差異：
  黑幫老大：增幅 ÷ 3，最低保底 +0.1（10%）
  黑幫幹部：增幅 ÷ 2
  黑幫小弟 / 市民 / 其他：完整計算

最終通緝值 = 現有通緝值 + 通緝增幅（無上限，可超過 100%）
```

**實際範例：**

| 剩餘刑期 | 市民/小弟 | 幹部 | 老大 |
|---|---|---|---|
| 30 分鐘 | +10% | +5% | +10%（保底） |
| 60 分鐘 | +20% | +10% | +10%（保底） |
| 120 分鐘 | +40% | +20% | +13.3% |
| 300 分鐘 | +100% | +50% | +33.3% |

**設計邏輯：**
- 剩餘刑期越長（罪行越重） → 越獄代價越大
- 黑幫老大/幹部享有折扣，但不再能透過短刑期零代價刷通緝值
- 老大至少有 10% 通緝代價（脫逃終究是違法的）

### 3.8 賭場系統

| 指令 | 說明 |
|---|---|
| `歐印` | 輸入賭局（依群組內通緝值決定查水表機率） |
| 各種牌局 | 梭哈、21點、輪盤等 |
| `+100`, `-一半` | 多人賭桌發牌前，可使用加減符號動態調整注碼 |
| `要`, `過`, `停` | 支援極度口語化的補牌與停牌動作 |

**查水表機制：**
- 賭局中的玩家通緝值越高 → 被警察突擊機率越大
- 有黑幫幹部以上在場（通緝值 ≥ 60%）→ 查水表機率減半
- 觸發查水表後，高通緝值玩家會被直接逮捕
